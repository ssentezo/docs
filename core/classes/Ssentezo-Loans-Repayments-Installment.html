Deletes an installment or installments of a given loan.

public static delete_intallments(DbAccess $db, int $loan_id[, int $installment_id = 0 ]) : int|string

If only the loan id is specified, all the installments will be deleted if the installment_id is specified the specified installment will be deleted

Parameters
$db : DbAccess

The database connection of the company

$loan_id : int

The id of the loan

$installment_id : int = 0

The id of the installment to delete, It's zero by default and in this case all installments will be deletes Note that these are soft deletes and can always be recovered

Return values
int|string —

If an error occurs, a string with the error description will be returned and If it's an integer it will indicate the records deleted.
