LeafAccount extends Account
in package
  • Application

LeafAccount.php : 11

The basic building block of the accounting system this account by default represents a leaf account i.e (NOT a category or sub category) So an instance of this class will be getting involved in transactions Meanwhile other classifications such as Categories and subcategories should extend this class But there is a Leafaccount class that is specifically crafed for transactions.
