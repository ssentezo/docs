public static createRepayment(DbAccess $db, array<string|int, mixed> $request, int $loan_id[, mixed $close_off = false ]) : int|string

A repayment process involves registering the repayment in the loan_repayments table and the using that money in form of active repayments to make full or partial installmets to the loan. An instance of the Repayment class will enable us record the repayment in the loan_repayments table. and It's from this table that we can retrive the different amounts and use them to make installmets
